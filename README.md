# BezKarla

Alternative frontend for OdKarla.cz product search powered by Luigi's Box API.

## Features

- Product search with pagination (96 results per page)
- Image gallery with hover/click thumbnail switching (fetched from OdKarla backend)
- Product detail modal with full-res images
- Sorting by relevance or price
- Links to OdKarla.cz and Amazon (when ASIN available)

## Usage

Open `index.html` in a browser.
